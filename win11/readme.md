# win11

## fileLink.bat

- 根据文件类型创建文件链接，硬链接，符号链接等
- 用来在不影响软件使用的情况下转移 C 盘文件
- [cmd/bat 文件 download link](https://github.com/lqfy-jhc/system-help-handbooks/blob/main/%E8%B5%A211/fileLink.bat)
